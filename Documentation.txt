This Is A Weather Analysis Project to Learn The Basicis Of Data Science And Python 

Through The Concepts Of Data Preprocessing And ML Techniques

Project Overview
 Title: Rainfall Prediction Using Random Forest Classifier

Objective
To build a machine learning model that predicts whether it will rain tomorrow based on historical weather data.

Problem Type
Binary Classification Problem

Target variable:
RainTomorrow

0 → No Rain

1 → Rain
Dataset Description

The dataset contains historical weather observations including:

Temperature Features:
MinTemp
MaxTemp
Temp9am
Temp3pm

Rain Features:
Rainfall
RainToday
RainTomorrow (Target)
Wind Features
WindGustSpeed
WindSpeed9am
WindSpeed3pm
WindGustDir
WindDir9am
WindDir3pm

Cloud & Humidity:
Humidity9am
Humidity3pm
Cloud9am
Cloud3pm

Other Features:
Location
Pressure9am
Pressure3pm
Sunshine
Evaporation


PROJECT ARCHITECTURE:

Raw Data
   ↓
Data Cleaning
   ↓
Feature Engineering
   ↓
Encoding
   ↓
Train/Test Split
   ↓
Random Forest Model
   ↓
Evaluation
   ↓
Model Saving (Pickle)
   ↓
Prediction System


 Final Output

The model predicts:

0 → No Rain Tomorrow 

1 → Rain Tomorrow
